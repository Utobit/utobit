QRGen

텍스트나 URL을 QR 코드로 즉시 변환합니다
클립보드 자동 감지 · PNG 저장 · 이미지 복사

다운로드

직접 다운로드 (.exe) 소스코드 (GitHub)

주요 기능

  • 텍스트 또는 URL 입력 → QR 코드 즉시 생성
  • 클립보드 자동 감지 — https:// URL 복사 시 자동 입력
  • QR 이미지 PNG 파일 저장
  • QR 이미지 클립보드 직접 복사
  • 단일 EXE — 설치 불필요, 실행 즉시 사용
