ColorScrap

트레이에서 동작하는 프로그램입니다
색상코드와 링크를 수집해 팔레트를 채워 재사용이 용이합니다

다운로드

직접 다운로드 (.exe) 소스코드 (GitHub)

단축키

Alt + C

커서 위치 색상 캡처

Alt + L

브라우저 URL + 타이틀 저장

Alt + Shift + C

마지막 색상 클립보드 복사

Alt + Z

팔레트 열기 / 닫기 (토글)

모든 단축키는 앱 설정에서 변경할 수 있습니다.

주요 기능

  • 화면 어디서나 색상 캡처 — HEX · RGB 즉시 확인
  • 브라우저 링크 캡처 — URL + 페이지 타이틀 자동 저장 (Chrome · Edge · Firefox)
  • 클릭 → 클립보드 복사 · 드래그 → 다른 앱에 바로 붙여넣기
  • 최근 10개 팔레트 자동 관리 · JSON 영구 저장
  • 단축키 자유 변경 (Ctrl / Alt / Shift + A–Z)
  • 트레이 상주 · 시스템 리소스 최소 사용
