QuickNote

트레이 상주 메모장. 노트는 .txt 파일로 로컬에 저장됩니다
다중 노트 · 목록 팝업 · 실시간 자동저장

다운로드

직접 다운로드 (.exe) 소스코드 (GitHub)

주요 기능

  • 노트를 개별 .txt 파일로 로컬 저장 — 파일 탐색기에서도 직접 접근 가능
  • 실시간 자동저장 (1.5초 디바운스) + Ctrl+S 즉시 저장
  • ≡ 버튼으로 노트 목록 팝업 — 빠른 전환
  • 삭제된 노트는 _trash 폴더로 이동 (자동 백업)
  • 저장 폴더 자유 지정 (기본: Documents/QuickNotes)
  • 트레이 상주 · 최소화 시 숨김 · 시스템 리소스 최소 사용
